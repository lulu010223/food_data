# Food Data Analysis

식품 데이터와 감미료 정보를 활용한 Python 데이터 분석 프로젝트.

## 주요 기능

- 제품명 기반 식품 검색
- 열량, 당류, 감미료 정보 조회
- 감미료별 장단점 검색
- 카테고리별 식품 조회
- 유사 제품 및 관련 제품 추천
- 데이터 시각화(Matplotlib)

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib

## 프로젝트 목적

식품에 포함된 당류 및 감미료 정보를 쉽게 확인할 수 있도록 프로그램을 구현함. 데이터 전처리, 검색 기능 구현, 시각화 과정을 통해 데이터 분석 역량을 향상시키는 것을 목표로 함.
